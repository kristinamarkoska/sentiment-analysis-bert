# Project Overview

This project uses a BERT transformer model to perform sentiment analysis (positive/negative) on Spanish movie reviews. The goal is to fine-tune a pre-trained BERT model to predict the sentiment of unseen reviews.

Dataset: 250 Spanish movie reviews

Task: Binary Classification (positive / negative)

Model: google-bert/bert-base-multilingual-cased

Libraries: transformers, datasets, torch, evaluate, scikit-learn
